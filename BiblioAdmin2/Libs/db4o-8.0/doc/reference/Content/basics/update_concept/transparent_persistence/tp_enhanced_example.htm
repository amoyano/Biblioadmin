TP Enhancement

Implementing the Activatable-interface manually is time consuming, repetitive and error-prone. Therefore it's desirable to automate this process. There are different ways to automatically enhance classes to support transparent persistence.

  • One possibility is to use a build-script which enhances the classes. For this a special Ant-task enhances the classes to support TPTransparent Persistence. See "Build Time Enhancement Example"
  • It's also possible to enhance classes at runtime. See "Load Time Enhancement"
